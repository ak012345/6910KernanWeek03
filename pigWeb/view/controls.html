Game Menu

Menu Bar

The menu bar enables the player to manipulate the game settings. Press and hold 'Alt + HOTKEY' to use the menu with the keyboard.

Game Submenu

Allows the player to exit the game. The player may use the Hotkey 'Ctrl + X' at any time within the game to quit.

Strategy Submenu

Allows the player to pick the strategy the computer will ise while playing it's turn. Use the hotKeys at any time during the game to change the current strategy.
